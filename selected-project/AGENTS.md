# AGENTS.md

## Project

This repository contains the planning documents and Laravel scaffold for an e-learning platform.

## Working Rules

- Keep architecture decisions documented in `docs/conception`.
- Keep Laravel application code inside `elearning-laravel`.
- Prefer small, reviewable changes with matching tests when behavior changes.
- Do not commit secrets. Use `.env.example` as the source of expected environment keys.

## Suggested Workflow

1. Read `IMPLEMENTATION_PLAN.md`.
2. Confirm the current phase in `docs/conception/phase_prompts.md`.
3. Implement in `elearning-laravel`.
4. Update docs when domain rules or architecture change.
